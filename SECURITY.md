# Security

## Supported Versions

The latest version of Tachyon is the supported version. You can find out what has been released [here](https://github.com/JasonBock/Tachyon/releases).

## Reporting a Vulnerability

If you think you have found a security vulnerability in Tachyon, please submit an issue [here](https://github.com/JasonBock/Tachyon/issues).
